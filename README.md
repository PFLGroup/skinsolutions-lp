# Skin Solutions — Landing Page

Premium, jednostronicowy landing page dla kliniki kosmetologii i medycyny estetycznej **Skin Solutions** (Mokotów, Warszawa).

**Live (GitHub Pages):** https://pflgroup.github.io/skinsolutions-lp/

## O projekcie
- Statyczna strona: `HTML + CSS + JS` (bez frameworka, bez build-stepu).
- Cel konwersyjny: rezerwacja wizyty przez Booksy + kontakt telefoniczny.
- Paleta marki: `#3a5d49` (głęboka zieleń) · `#efedeb` (krem) · `#b0c9aa` (szałwia).
- Treść pozyskana ze strony klienta https://skinsolutions.pl (czerwiec 2026).

## Struktura
```
index.html              # strona
assets/css/styles.css   # system projektowy + style
assets/js/main.js       # interakcje (nav, scroll-reveal, licznik)
assets/img/             # zasoby graficzne
robots.txt, sitemap.xml # SEO
```

## Hosting
GitHub Pages (branch `main`, katalog `/root`). Ścieżki względne — działa pod ścieżką projektu `/skinsolutions-lp/`.
